# UPLOAD.NET

UPLOAD.NET is a .NET 8.0 Web API designed to provide a SaaS-like image management solution. Users can create accounts, log in, and utilize the API to upload images to the root folder and retrieve all uploaded images. It functions similarly to services like MinIO.

Features

User Authentication:

Create users via a secure registration process.

Log in with credentials to access protected API endpoints.

Image Management:

Upload images directly to the root folder of the API.

Retrieve all uploaded images.

Scalable Design:

Built with .NET 8.0 for optimal performance and scalability.

Prerequisites

.NET SDK 8.0 or later

SQL Server or any supported database for user management

IDE (e.g., Visual Studio 2022)
